# Cadence site map

Where everything lives in the app.

## Top navigation

| Item | What it does |
|------|----------------|
| **Logo** | Back to Dashboard (or homepage if signed out) |
| **Sign out** | End cloud session |
| **Notifications** | SEO score drops, reminders |
| **+ New audit** | Run a fresh URL analysis → saves to Campaign History |
| **Profile pill** | Your marketer profile (name, avatar color) |
| **Brand dropdown** | Switch between client/site workspaces |
| **Stack** | Jump to Kerygma, CitePilot, Aegis, Studio hub (if enabled) |
| **Mobile menu** | Same nav on small screens |

## Main views

### Dashboard

Your home base after the first audit.

- Brand summary and readiness score
- Strengths, gaps, opportunities
- **Create** cards — generate keywords, blog, social, email, lead magnet
- Link to SEO Agent and Campaign History
- Growth stack insights (if connected)

### SEO Agent

Full-site SEO workspace.

| Tab | Purpose |
|-----|---------|
| Overview | Scores, executive summary, PDF export |
| Technical crawl | Page-by-page issues |
| Keyword gaps | Terms you’re missing |
| Meta tags | Rewritten titles & descriptions |
| 100% roadmap | Prioritized fix list |
| Content plan | **Topics to write** (ideas only) |
| Competitors | Compare another domain |
| Uplift | Track improvements over time |
| SEO stack | Google Search Console, GA4, CitePilot citations |

### Campaign Studio

Opens when you generate or open an asset.

- Live editor + AI refine tools
- SEO checklist (blog)
- **Copy WordPress HTML** / Publish (blog)
- Image generation (Imagen)
- Social previews (social posts)
- Version history within the session

### Campaign History

Saved audit runs and past campaign versions.

- Restore an older run
- Compare drafts
- Export campaign ZIP

### Studio

Growth stack hub — bundles, connected products, billing links (when stack UI is on).

### Document Scans

Upload PDFs/docs for AI marketing analysis (separate from site audits).

### Marketer Profile

Your name, role, and how Cadence addresses you in copy.

### Settings

| Tab | Purpose |
|-----|---------|
| General | Newsletter / Kit URL, agency name |
| Integrations | WordPress, GSC, GA4, growth stack API keys |
| Team | Members, client brands (Team plan) |
| Billing | Pro / Team / bundles |
| Studio | Link Kerygma, CitePilot, Aegis accounts |
| Cloud | Data & account |

## Two things people confuse

### Brand dropdown vs Campaign History

| | Brand dropdown | Campaign History |
|--|----------------|------------------|
| **Purpose** | Switch **client sites** | Browse **saved audit runs** |
| **When to use** | Agency with multiple URLs | “Load my audit from last week” |

### SEO Agent “Content plan” vs Dashboard “Blog post”

| | Content plan (SEO Agent) | Blog post (Dashboard) |
|--|--------------------------|-------------------------|
| **Output** | Topic list + keywords | Full article draft |
| **Publish?** | No — it’s a brief | Yes — WordPress export |

---

## Typical weekly workflow

1. **Dashboard** — check strategy
2. **SEO Agent** — run or refresh audit
3. Pick a topic from **Content plan**
4. **Dashboard** → generate **Blog post**
5. **Campaign Studio** → edit → **Copy WordPress HTML**
6. WordPress → paste → featured image → publish
7. **Kerygma** (optional) — social distribution
