## Quick answers

**I just want to publish a blog post.**  
→ [How to post](./HOW_TO_POST.md) — Method A (copy/paste) works on the free plan.

**Where do saved audits live?**  
→ **Campaign History** in the top nav (not the brand dropdown).

**How do I switch between client sites?**  
→ Top-right **brand dropdown** (building icon). Each site is a separate workspace.

**How do I add a newsletter signup to my blog?**  
→ **Settings → General** → paste your Kit (ConvertKit) form URL, then export a **blog post** (not SEO playbook).

**Cadence vs Kerygma vs CitePilot?**  
→ Cadence = strategy + SEO + content drafts. Kerygma = social autopilot. CitePilot = AI citation tracking.
